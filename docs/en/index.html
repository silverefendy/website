Project Overview

A high-level guide to the marketplace platform, its features, architecture, and documentation map.

Introduction

The application is a multi-role marketplace for administrators, sellers, and customers. It combines a React frontend, Express API, and MySQL database to support product discovery, shopping workflows, seller analytics, and platform administration.

Features

  • Customer browsing, wishlist, reviews, cart, checkout, and order history.
  • Seller dashboard, product views, stock warnings, and sales analytics.
  • Administrator pages for users, categories, orders, and site settings.
  • JWT authentication, refresh tokens, upload support, and MySQL constraints.

Technology Stack

FrontendReact 18, Vite, TailwindCSS, Zustand, React Router, Axios
BackendNode.js, Express, JWT, bcrypt, Multer, Joi, Helmet, CORS
DatabaseMySQL with foreign keys, indexes, check constraints, and seed data
