Production Deployment

Deploy the marketplace to a VPS or container platform with Nginx, SSL, PM2, Docker, and backups.

VPS Deployment

Provision Ubuntu 22.04 or 24.04, install Node.js, MySQL, Nginx, PM2, UFW, and Certbot, then clone the repository to /var/www/marketplace.

Docker Deployment

Use docker-compose.yml to run MySQL, backend, and frontend. Replace demo secrets, use persistent volumes, and avoid storing production credentials in Git.

Nginx

Nginx serves frontend/dist, proxies /api to Express, and proxies /uploads to the backend upload route.

SSL

sudo certbot --nginx -d example.com -d www.example.com

Verify DNS before requesting certificates.

PM2

pm2 start ecosystem.config.js --env production
pm2 save
pm2 startup systemd

Domain Setup

Create A records for root and www hostnames pointing to the VPS IP address.

Backup Strategy

Run daily MySQL dumps, archive uploads, store off-server copies, and test restore procedures monthly.
