Troubleshooting

Problem-solving reference for installation, build, database, network, SSL, and PM2 issues.

Installation Errors

Check Node.js version, npm availability, filesystem permissions, and network access to npm registry.

Build Errors

If Vite is missing, run npm install in the frontend directory and ensure package files are synchronized.

Database Errors

Verify MySQL service, database name, user grants, password, host, and port. Test direct CLI login.

Network Errors

CORS failures usually require updating ALLOWED_ORIGINS. Proxy failures require checking Nginx and PM2 backend port.

SSL Errors

Confirm DNS records, firewall ports 80/443, Nginx syntax, and Certbot domain names.

PM2 Errors

Use pm2 status, pm2 logs marketplace-api, and pm2 restart marketplace-api.
