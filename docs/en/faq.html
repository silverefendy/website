Frequently Asked Questions

Common questions, solutions, and best practices for operating the marketplace.

Common Questions

Typical questions cover supported roles, required database, upload location, production readiness, and default seed accounts.

Common Solutions

  • Run database imports before using the app.
  • Set CORS to the exact frontend origin.
  • Use strong JWT secrets.
  • Back up uploads and database together.

Best Practices

Use staging before production, never commit secrets, keep package locks synchronized, test migrations, and monitor logs.
