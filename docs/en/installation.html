Installation Guide

Install the project on Windows, Ubuntu, or Debian and verify local development services.

Requirements

  • Git for source control.
  • Node.js 20 LTS and npm.
  • MySQL 8.0 or compatible MariaDB.
  • Terminal access and permission to create databases.

Windows Installation

Install Git from git-scm.com, Node.js LTS from nodejs.org, and MySQL Community Server. Verify with:

git --version
node -v
npm -v

Ubuntu Installation

sudo apt update
sudo apt upgrade -y
sudo apt install -y curl git mysql-server
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

Debian Installation

Install default MySQL server packages, build tools, Git, and NodeSource Node.js packages. Debian users may receive MariaDB through default packages; confirm compatibility before production use.

Environment Setup

Create backend/.env for database, JWT, CORS, uploads, and WhatsApp variables. Create frontend/.env for Vite variables such as API URL, site name, currency, and support contact.

Verification

mysql -u marketplace_user -p marketplace < schema.sql
mysql -u marketplace_user -p marketplace < seed.sql
cd backend && npm run dev
cd frontend && npm run dev

Open the API health endpoint and frontend URL.
