Panduan Deployment Production

Panduan penerapan di VPS atau container dengan Nginx, SSL, PM2, Docker, domain, dan strategi backup.

Penerapan VPS

Siapkan Ubuntu 22.04 atau 24.04, instal Node.js, MySQL, Nginx, PM2, UFW, dan Certbot, lalu clone repository ke /var/www/marketplace.

Deployment Docker

Gunakan docker-compose.yml untuk menjalankan MySQL, backend, dan frontend. Ganti secret demo, gunakan volume persisten, dan jangan menyimpan kredensial production di Git.

Nginx

Nginx menyajikan frontend/dist, meneruskan /api ke Express, dan meneruskan /uploads ke backend.

SSL

sudo certbot --nginx -d example.com -d www.example.com

Pastikan DNS benar sebelum membuat sertifikat.

PM2

pm2 start ecosystem.config.js --env production
pm2 save
pm2 startup systemd

Pengaturan Domain

Buat A record untuk domain utama dan www yang mengarah ke IP VPS.

Strategi Backup

Jadwalkan dump MySQL harian, arsip upload, simpan salinan di luar server, dan uji restore setiap bulan.
