Pertanyaan yang Sering Diajukan

Pertanyaan umum, solusi umum, dan praktik terbaik untuk mengoperasikan marketplace.

Pertanyaan Umum

Pertanyaan umum biasanya terkait role yang tersedia, database yang dibutuhkan, lokasi upload, kesiapan production, dan akun seed default.

Solusi Umum

  • Import database sebelum menggunakan aplikasi.
  • Atur CORS sesuai origin frontend.
  • Gunakan JWT secret yang kuat.
  • Backup upload dan database secara bersamaan.

Praktik Terbaik

Gunakan staging sebelum production, jangan commit secret, jaga package lock tetap sinkron, uji migrasi, dan pantau log.
