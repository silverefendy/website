Pemecahan Masalah

Referensi penyelesaian masalah instalasi, build, database, jaringan, SSL, dan PM2.

Error Instalasi

Periksa versi Node.js, ketersediaan npm, permission filesystem, dan akses jaringan ke registry npm.

Error Build

Jika Vite tidak ditemukan, jalankan npm install di direktori frontend dan pastikan package file sinkron.

Error Database

Periksa service MySQL, nama database, grant user, password, host, dan port. Uji login langsung via CLI.

Error Jaringan

Kegagalan CORS biasanya perlu pembaruan ALLOWED_ORIGINS. Kegagalan proxy perlu pengecekan Nginx dan port backend PM2.

Error SSL

Pastikan DNS, firewall port 80/443, syntax Nginx, dan nama domain Certbot sudah benar.

Error PM2

Gunakan pm2 status, pm2 logs marketplace-api, dan pm2 restart marketplace-api.
